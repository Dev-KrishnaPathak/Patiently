## 🎨 Visual Features

### Info Icon States:

**Before Hover**:
- Emoji: ℹ️
- Opacity: 60% (subtle gray)
- Size: Normal (1rem)
- Cursor: Help cursor (question mark)

**On Hover**:
- Emoji: ℹ️
- Opacity: 100% (full brightness)
- Size: 110% (slightly bigger)
- Tooltip: Visible with fade-in animation
